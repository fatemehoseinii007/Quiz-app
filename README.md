# Quiz App 🧠

An interactive quiz application built with HTML, CSS, Tailwind, and pure JavaScript.

## Features
- Multiple choice questions
- Automatic score calculation
- Fully responsive design (mobile & desktop)
- Smooth animations
- Instant feedback on answers

## How to Run
Open `src/index.html` in your browser.

## Project Structure
Quiz-app/
├── src/
│ ├── index.html
│ ├── style.css
│ └── script.js
└── README.md


## Technologies
- HTML5
- CSS3
- Tailwind CSS
- JavaScript (ES6)
